Priyanshu — Front Page Portfolio

A clean, single-page front-page portfolio built with HTML, CSS and JavaScript.
Designed for smooth full-screen scrolling (scroll-snap), subtle animations, and a floating dot navigation to showcase: Home, About, Experience, Skills, Projects, Education, Contact. Perfect as a lightweight personal portfolio to host on GitHub Pages.

Features

Single-file (index.html) portfolio — no build step required

Full-screen scroll sections using CSS scroll-snap

Floating navigation dots with labels and active state

IntersectionObserver-driven entrance animations

Animated skill bars that fill when the Skills section enters view

Experience timeline, projects grid, education and contact form

Contact & signup demo store submissions in localStorage (easy to replace with API)

Responsive — works on desktop and mobile

No external libraries — small and easy to customize

Demo

Open index.html in any modern browser (Chrome / Firefox / Edge / Safari).
(For GitHub Pages: push the repository and enable Pages from the main branch — the index.html will be served automatically.)
